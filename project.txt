library {
  하나의 공간에 모든 라이브러리 파일들을 만듭니다.
  다른 프로젝트에서 해당 라이브러리를 사용합니다.
  내 라이브러리는 따로 명시하지 않아도 글로벌하게 default로 추가되면 좋겠는데
}

프로젝트 관리 {
  작성한 코드가 프로젝트와 밀접한 경우 프로젝트 디렉토리 밑에 둔다.
  작성한 코드가 프로젝트와 관련이 있으나 프로젝트의 특성을 타지 않고 제너럴한 경우 일반 코드 디렉토리에 넣는다.
}
